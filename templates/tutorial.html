Tutorial

Welcome to the EEG streaming web application! Follow the steps below to set up and use the platform.

1. Setup and Access

Before using the application, ensure that your EEG device is properly connected and configured:

  • Connect your EEG device to your computer via the appropriate interface (USB, Bluetooth, etc.).
  • Ensure that all necessary drivers and software for your device are installed.
  • Download websocket from the Setup page under "Download"
  • Start the EEG data streaming executable on your device.
  • Follow the instructions on the Setup page to ensure all physical and digital tools are functional.

2. Navigating the Interface

The web application consists of the following pages:

  • Home Page: Displays real-time EEG readings and visualizations.
  • About Page: Provides background information on the project.
  • Tutorial Page: This page, which guides users through setup and usage.
  • Setup Page: Assists in configuring the EEG device and connection.

3. Real-Time EEG Streaming

Once the connection is established, the app will begin displaying live EEG data:

  • The application will attempt to connect to the EEG data source via WebSocket.
  • Once connected, EEG signals will be processed and displayed in real-time.
  • Graphical representations of electrode activity will update dynamically.
